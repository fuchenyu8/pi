# CMC24 - 物流分拣中心货量预测与排班优化

## 项目概述

本项目针对物流分拣中心的货量预测和人员排班优化问题，采用多种机器学习和优化算法进行求解。

## 目录结构

```
cmc24/
├── MATLAB/
│   ├── data_process.m              # 数据预处理（附件1、附件2）
│   ├── main1_LSTM日货量预测.m       # LSTM神经网络日货量预测
│   ├── main2_ARIMA小时货量预测.m    # ARIMA模型小时货量预测
│   └── main3_问题二线路修正预测.m   # 线路修正货量预测
├── Python/
│   ├── 问题三_线性规划排班.py       # 线性规划排班优化
│   └── 问题四_模拟退火算法.py       # 模拟退火算法框架
└── README.md
```

## 问题说明

### 问题一：货量预测
- **日货量预测**：使用LSTM神经网络预测未来30天各分拣中心的日货量
- **小时货量预测**：使用ARIMA模型预测未来30天各分拣中心的小时货量

### 问题二：线路修正
- 根据新线路数据修正货量预测结果
- 考虑不同分拣中心的费率调整

### 问题三：排班优化
- 使用线性规划方法进行人员排班
- 优化固定员工和临时员工的配置

### 问题四：调度优化
- 使用模拟退火算法求解复杂的调度问题
- 考虑出勤率、连续上班、货量等约束条件

## 运行环境

### MATLAB
- MATLAB R2020a 或更高版本
- 需要 Deep Learning Toolbox（LSTM部分）
- 需要 Econometrics Toolbox（ARIMA部分）

### Python
- Python 3.8+
- 需要安装以下库：
  ```bash
  pip install pulp pandas numpy
  ```

## 使用说明

### MATLAB代码
1. 将附件数据文件放在MATLAB工作目录
2. 首先运行 `data_process.m` 进行数据预处理
3. 运行 `main1_LSTM日货量预测.m` 进行日货量预测
4. 运行 `main2_ARIMA小时货量预测.m` 进行小时货量预测
5. 运行 `main3_问题二线路修正预测.m` 进行线路修正

### Python代码
1. 确保已安装所需依赖库
2. 将 `结果表4.csv` 放在Python脚本同目录
3. 运行 `问题三_线性规划排班.py` 进行排班优化
4. 运行 `问题四_模拟退火算法.py` 进行调度优化

## 输出结果

- `结果表1.csv`：日货量预测结果
- `结果表2.csv`：小时货量预测结果
- `结果表3.csv`：线路修正后日货量
- `结果表4.csv`：线路修正后小时货量
- `排班结果.csv`：排班优化结果
